## Homework on Laravel

Общие сведения:
Вам необходимо создать систему учета сотрудников в компании.
Если вы не можете выполнить какой-то пунктов, ничего страшного. При сдаче результатов опишите проблемы с которыми вы столкнулись.
Тестовое задание обязательно должно находиться на удалёном репозитории (gitlab\github\bitbucket).
Стили для Front-end являются необязательной частью, но при желании вы можете использовать любой css фреймворк или написать свои стили. Использование стилей вам даст дополнительный плюс. 

Технические требования:
PHP 7.1+
Laravel 5.7+
MySQL 5.6+	

База данных: 
	Нужно создать базу данных, в которой обязательно должны быть следующие таблицы: компании, сотрудники, должности. Также у сотрудника должна быть зарплата, вы можете её реализовать внутри должности или сотрудника, по вашему усмотрению. Остальные     атрибуты и записи вы можете сделать по своему усмотрению.

Задание:
Создать проект laravel.
Создать миграции.
Создать Seeds.
Сделать страничку с выводом компаний и их сотрудников с должностями и зарплатами. 
Добавить возможность перейти в профиль каждой компании. 
В профиле можно Добавить\Изменить описание.
В профиле можно Добавить\Изменить фото.
Сделать авторизацию для компании.
Компания в своём профиле может Создать\Удалить\Изменить информацию о сотруднике.
 Компания может изменить должность сотрудника.
 В профиле компания может удалить себя. (Связи удалить каскадно)
 В профиле компаний сделать поле для комментариев.
 Авторизованная компания может оставлять комментарий в профиле любой компании. 
 Комментарии видно только авторизованным компаниям.
